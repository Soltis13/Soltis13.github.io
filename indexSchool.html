Welcome to my Web Development Bootcamp Site.

My name is Scott Soltis and this is webpage covers the University of Minnesota Bootcamp Program. This program is a 6 month full stack education covering technologies from browser based (Javascript, jQuery, Bootstrap), to server side development(Node, Express, MySQL, MongoDB), as well as deployment(git, heroku).

Prioir to joining this program I worked as an Engineer designing and manufacturing varrious Medical Device technologies.

My passions involve developing technology that delivers high end user experiences. I've found that technology is great for improveing human experiences but only if it is designed to be invisable and complimentary to the situation. Poor design only leads to headache.
